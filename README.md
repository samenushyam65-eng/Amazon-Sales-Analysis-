📌 Project Overview
This project presents an interactive Amazon Sales Analysis Dashboard built using Microsoft Power BI. It transforms raw Amazon sales transactional data into meaningful business insights across revenue trends, regional performance, payment behavior, and product-level benchmarking — enabling data-driven decision-making for business stakeholders.

🎯 Problem Statement
Amazon generates vast volumes of sales data across multiple product categories, regions, and payment methods. Without a structured analytics system, it becomes difficult to track performance, identify opportunities, and optimize pricing strategies. This dashboard addresses that gap by providing a centralized, interactive reporting solution.

💡 Features

📈 Revenue & profit trend analysis over time
🗺️ Regional revenue distribution using map visuals
💳 Payment method breakdown and discount impact analysis
🛒 Top product rankings by ratings and quantity sold
🎛️ Dynamic slicers for filtering by category, region, and time
🔘 Navigation buttons for seamless page transitions
